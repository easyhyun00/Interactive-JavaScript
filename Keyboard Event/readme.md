# 인터렉티브 자바스크립트

## [키보드 이벤트]

### [ KeyboardEvent.type ]

#### keydown

키보드 버튼을 누른 순간

#### keypress

키보드 버튼을 누른 순간 <br>
(출력값이 변하는 키에서만 동작 => 기능적인 역할 하는 키에서는 동작X) <br>
(하나의 키 계속 누를 때, 한 번만 동작) <br>
(영어가 아닐 때, 반응하지 않는 경우 O) <br>
=> 웹 표준에서 권장하는 방법 X

#### keyup

키보드 버튼을 눌렀다 뗀 순간

#### KeyboardEvent.key

이벤트가 발생한 버튼의 값

#### KeyboardEvent.code

이벤트가 발생한 버튼의 키보드에서 물리적인 위치
